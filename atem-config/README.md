# ATEM Software Control Configuration Files

This folder contains configuration files and saved setups for the Blackmagic ATEM Software Control application used with the church's 1 M/E Production Switcher 4K.

## Purpose

These configuration files allow you to:
- Back up switcher settings and macros
- Review and modify configurations remotely
- Restore settings after software updates or reinstallation
- Share configurations between operators

## File Types

ATEM Software Control typically saves files with these extensions:
- `.xml` - Switcher configuration and settings
- `.atem` - Saved switcher states (if applicable)
- Macro files and custom setups

## Windows PC Location

ATEM Software Control is installed at:
```
C:\Program Files (x86)\Blackmagic Design\Blackmagic ATEM Switchers\ATEM Software Control
```

## Usage

1. Export/save configurations from ATEM Software Control
2. Copy files to this folder
3. Commit and push to GitHub for remote access
4. To restore: download files and import into ATEM Software Control
